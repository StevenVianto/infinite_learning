# Tugas Dart Dasar - Fungsi Sederhana

## Deskripsi
Project ini adalah latihan dasar Dart yang saya kerjakan untuk memahami cara membuat fungsi.
Di dalamnya ada beberapa contoh fungsi dengan jenis yang berbeda, mulai dari fungsi tanpa return sampai fungsi yang mengembalikan nilai.

## Tujuan Pembelajaran
- Memahami perbedaan fungsi dengan dan tanpa parameter.
- Memahami fungsi yang mengembalikan nilai (return).
- Mencoba membagi kode ke beberapa file agar lebih rapi.

## Fitur yang Dibuat
1. Fitur sapaan
- `sapaUmum()` -> menampilkan sapaan umum.
- `sapaNama(String nama)` -> menampilkan sapaan berdasarkan nama.

2. Kalkulator luas persegi
- `luasPersegi(double sisi)` -> menghitung luas persegi dengan rumus `sisi x sisi`.

3. Kalkulator diskon
- `hitungDiskon(double hargaAsli, double persenDiskon)` -> menghitung nilai diskon dan harga akhir.

## Struktur File
- `greetings.dart`
- `luas_persegi.dart`
- `diskon.dart`
- `main.dart`

## Cara Menjalankan
1. Pastikan Dart SDK sudah ter-install.
2. Buka terminal di folder project.
3. Jalankan perintah berikut:

```bash
dart run main.dart
```

## Catatan
Kode ini masih sederhana karena fokusnya untuk latihan konsep dasar fungsi.
Ke depannya bisa dikembangkan lagi, misalnya input dari user dan validasi yang lebih lengkap.
